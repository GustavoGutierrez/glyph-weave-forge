# Theme and style reference

This document captures practical style targets for the built-in themes.

## Base defaults

- Body size: 11–12 pt
- Neutral margins suitable for A4/Letter documents
- High-contrast text on light background
- Clear heading hierarchy (H1/H2/H3)

## Built-in theme intents

### Invoice

- Fast scanning and tabular readability
- Sans-serif emphasis, compact spacing
- Strong visual separation for totals

### ScientificArticle

- Dense, formal, citation-friendly layout
- Serif-forward typography and restrained styling
- Predictable spacing for long-form technical content

### Professional

- Executive readability and balanced whitespace
- Neutral palette with restrained accent use
- Clear section and bullet hierarchy

### Engineering

- Precision-first presentation for technical material
- Strong code/table/diagram legibility
- Conservative styling with explicit structure

### Informational

- Guide/tutorial readability with larger body targets
- Friendly spacing and approachable hierarchy
- Emphasis through structure rather than heavy decoration

## Constraints and honesty

Current theme controls focus on supported properties (fonts, sizing, margins,
heading scale, and palette). Advanced controls such as multi-column tuning,
line-height policy, and paragraph alignment profiles are intentionally limited and
should not be advertised beyond implemented behavior.
